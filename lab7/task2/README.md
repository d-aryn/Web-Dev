# OOP Python Demo (Animal / Dog / Cat)

This project demonstrates:
- A base class (`Animal`) with `__init__`, instance methods, and `__str__`
- Inheritance (`Dog`, `Cat`) with unique attributes/methods
- Method overriding (`speak()` and `describe()`)
- Polymorphism by calling `speak()` on a list of mixed object types
- Simple module organization (`models.py` + `main.py`)

## Run

```bash
python3 main.py
```
